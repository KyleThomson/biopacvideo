Biopacvideo
The Biopacvideo repository is a comprehensive collection of software tools designed to handle tethered and telemetry data for laboratory animals. The suite consists of three main programs:

Biopac Video: Used for recording up to 16 tethered or telemetry EEG channels along with video feeds, providing synchronized data capture for detailed analysis, and automated feeding handling.
Seizure Playback: Allows users to review EEG recordings, view the associated video feeds, identify seizures, and rate them on the racine scale.
Project Manager: A program designed to create and manage reviewed files for a single experiment, helping to organize and analyze experimental data.
